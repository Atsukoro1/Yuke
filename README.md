# Yuke - Clean and fast Messenger-like chat app

## Installation

1. Clone this repo 
2. Install dependencies using "npm install"
3. Fill the dotenv variables like this

```bash
JWTSECRET=somerandomsecret
MONGOURI=yourmongodburi
PORT=port
```

## Usage

Now you can run Yuke using: 
```bash
npm run dev 

or

node index.js
```
